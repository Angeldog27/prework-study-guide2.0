Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The ul element stands for unordered lists and goes with li elements to create bullet points

CSS

  • Css stands for cascading style sheets, and helps add the styling for the webpage
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • * selects all elements to recieve the value

Git

  • Git is a version control system that tracks changes in any set of computer files, used for coordinating work among programmers collaboratively developing source code during software development.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is how the user is able to interact with the webpage
  • Control+Shift+I opens inspect from chrome developer tools
  • x++ is a shortcut way of writing x+1 in for loops
  • /* before and */ after the block of code To comment out multiple lines of JavaScript code
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
